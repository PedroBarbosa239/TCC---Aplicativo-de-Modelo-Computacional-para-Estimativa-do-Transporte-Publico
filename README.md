#  Modelo Computacional para Estimativa de Transporte Público

Sistema inteligente para estimativa da **localização de ônibus** e **tempo de chegada** sem uso de GPS, utilizando análise de dados históricos, condições de tráfego, clima e colaboração dos usuários.

---

##  Sobre o Projeto

Este projeto foi desenvolvido como Trabalho de Conclusão de Curso (TCC) com o objetivo de melhorar a experiência no transporte público em cidades que **não possuem rastreamento em tempo real**.

A proposta é utilizar técnicas de **ciência de dados e aprendizado de máquina** para prever a posição dos ônibus, mesmo com dados incompletos.

---

##  Tecnologias Utilizadas

- Python
- Pandas
- Scikit-learn
- APIs externas (trânsito e clima)
- React Native
- Expo

---

##  Como Funciona

O sistema realiza previsões com base em:

-  Dados históricos de horários de ônibus  
-  Condições de trânsito em tempo real  
-  Condições climáticas  
-  Feedback dos usuários  

### Modelos utilizados:
- Regressão
- Séries temporais
- Agentes Inteligentes

### Métricas de avaliação:
- MAE (Erro Médio Absoluto)
- RMSE (Raiz do Erro Quadrático Médio)

---

##  Impacto

- Melhoria da mobilidade urbana  
- Redução da incerteza no transporte público  
- Aplicação prática de IA em cenários com dados incompletos
